React Web App Google API

Финансовый учет (Web App)

Разработка мобильного веб-приложения (SPA) для автоматизации учета приходов и расходов компании. Проект создан для замены неудобных отчетов в мессенджерах: теперь сотрудники вносят данные через удобный интерфейс, а информация структурированно сохраняется в облачной БД с моментальным уведомлением руководства.

Подробнее о проекте

🛠 Стек технологий:

  • Frontend: React, Tailwind CSS, HTML/JS.
  • Backend & БД: Google Apps Script, Google Sheets API, Google Drive API.
  • Интеграции: Telegram Bot API.
  • Хостинг: GitHub Pages.

🎯 Ключевые функции:

  • ▹ Авторизация: Вход по паролю с привязкой сотрудника к компаниям.
  • ▹ Умная форма: Выбор типа операции, категории, автоформат дат и загрузка чеков.
  • ▹ TG-уведомления: Мгновенная отправка отчетов руководителю через бота.
  • ▹ Сохранение файлов: Конвертация (Base64) и загрузка чеков на Google Drive.
  • ▹ Mobile-first & CRUD: Адаптив, кэширование (localStorage), редактирование данных по ID без дублей.

📈 Ценность для бизнеса:

Проект полностью решил проблему потери чеков и финансовой путаницы в чатах. Сбор данных автоматизирован, а руководство получило прозрачный контроль за наличными средствами в реальном времени.

TimeFlow Cyber Edition

Интеллектуальный тайм-трекер в стиле «кибер-реализм» для глубокого анализа продуктивности. Приложение не просто замеряет время, а превращает его в структурированные данные, автоматически синхронизированные с облачной экосистемой Google.

Подробнее о проекте

🛠 Стек технологий:

React, Tailwind CSS, Google Cloud API (Calendar & Tasks), Lucide Icons.

🎯 Ключевой функционал:

  • ▹ Двусторонняя синхронизация: Завершенные сессии улетают в Google Calendar, а задачи импортируются из Google Tasks одним нажатием.
  • ▹ Глубокая аналитика: Мониторинг плана (160 часов), динамический годовой график и детализация по проектам (ядрам).
  • ▹ Управление данными: Ручная корректировка, динамическое создание категорий и продвинутый CSV-импорт (Toggl Track).
  • ▹ Cyberpunk UI/UX: Адаптивный неоновый интерфейс с «мини-режимом» для фокусировки и поддержкой горячих клавиш.

🏆 Мои достижения в проекте:

  • ▹ Сложные интеграции: Успешно реализована авторизация OAuth2 и работа с Google Calendar API v3 / Tasks API.
  • ▹ Data Parsing: Написан мощный алгоритм для преобразования сырых отчетов CSV в структурированную базу логов.
  • ▹ Архитектура состояния: Приложение мгновенно пересчитывает статистику за весь год при переключении фильтров, обрабатывая огромный массив данных без зависаний.

Система учета финансов

ТГ-бот + Google Sheets

Задача

Избавить клиента от ручного ведения расходов в блокнотах и неудобных приложениях.

Решение

Разработан ТГ-бот, который парсит текстовые сообщения с тратами и мгновенно раскидывает их по категориям в защищенной Google Таблице с авто-дашбордом.

Результат

Время на отчетность сократилось с 1 часа в неделю до 2 минут в день. Данные всегда под рукой.
