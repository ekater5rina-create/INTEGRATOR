Система учета финансов

ТГ-бот + Google Sheets

Задача

Избавить клиента от ручного ведения расходов в блокнотах и неудобных приложениях.

Решение

Разработан ТГ-бот, который парсит текстовые сообщения с тратами и мгновенно раскидывает их по категориям в защищенной Google Таблице с авто-дашбордом.

Результат

Время на отчетность сократилось с 1 часа в неделю до 2 минут в день. Данные всегда под рукой.

Cyber/Neon UI Kit

Figma + Tilda Zero Block

Задача

Создать единую дизайн-систему для технологичного стартапа, чтобы ускорить сборку новых лендингов.

Решение

Отрисован UI Kit в темной теме с неоновыми акцентами: кнопки, формы, карточки, типографика. Все элементы перенесены в библиотеку Tilda.

Результат

Сборка новых страниц стала занимать в 2 раза меньше времени, дизайн стал консистентным.
